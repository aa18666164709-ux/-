# 小红书虚拟产品 OPC

这个仓库用于沉淀小红书虚拟产品内容工厂的规则、知识库和 Codex 执行提示词。

## 当前阶段

第一阶段先迁移并跑通：**科目一口诀资料项目**。

核心目标：

1. 让 Codex 读取仓库规则，自动生成科目一挂车笔记。
2. 每篇输出标题、正文、标签、主图文案、主图生成提示词。
3. 每篇自动评分和风险检查。
4. 后续再接入审核表、主图生成、发布助手和数据回流。

## 目录结构

```text
knowledge_base/kemuyi/        科目一项目知识库
prompts/kemuyi/               Codex 执行提示词
outputs/                      后续生成内容输出目录
```

## 使用方式

让 Codex 执行：

```text
读取 knowledge_base/kemuyi/ 和 prompts/kemuyi/，按科目一项目规则生成今天的科目一挂车笔记。
```
