# 科目一正文结构库

## 1. 临考救命型
临考焦虑 → 先安抚 → 题多但重点集中 → 建议先看重点 → 引导商品。

## 2. 卡80多分反转型
总卡80多分 → 不是不努力 → 是复习顺序错了 → 先理易混点 → 引导商品。

## 3. 扣分罚款专项型
扣分罚款老混 → 不要靠感觉选 → 先把规则整理出来 → 再刷题会顺很多。

## 4. 标志标线专项型
标志题容易懵 → 不是死记图片 → 要先分类型和场景 → 引导资料。

## 5. 真实吐槽型
科一真的烦 → 题多又杂 → 但不是没规律 → 把重点集中看 → 引导商品。

## 6. 懒人速记型
不想一套套刷 → 先抓高频点 → 再刷错题/模拟 → 引导商品。

## 7. 备考流程型
先过重点 → 再刷专项 → 再看错题 → 最后模拟 → 引导商品。

## 8. 还愿反馈型
刚考完/终于过了 → 幸好考前看了重点 → 分享给同样临考的人 → 引导商品。
