# 科目一主图结构库

## 1. 成绩截图型
上方考试成绩页面，下方大字标题。

适合突出：

- 结果感
- 临考焦虑
- 卡分突破

## 2. 极简大字型
浅色背景，大字痛点。

适合快速抓点击。

## 3. 资料实拍型
资料、打印纸、学习桌场景。

适合增强真实感。

## 4. 手机截图感
像手机页面、备忘录、文件列表。

适合轻内容、真实分享感。

## 5. 专项重点型
扣分罚款、标志题、易混题等重点提示。

适合专项痛点。

## 6. 临考救命型
时间焦虑 + 大字提醒。

适合：

- 这两天考试
- 考前一晚
- 临考没底的人

## 禁止风格
主图不能像：

- PPT
- 宣传单
- 电商详情页
- 满屏小字
- 廉价营销图
