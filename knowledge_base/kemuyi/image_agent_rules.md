# 科目一主图 Agent 规则

## 主图比例
小红书 3:4 竖版。

## 主图目标
只负责提高点击率，不负责详细介绍产品。

## 主图风格
真实、极简、留白多、像小红书爆款封面。

不要像：

- 电商详情页
- PPT
- 宣传单
- 满屏小字
- 廉价营销图

## 主图文案要求
1. 短、狠、直给。
2. 默认2行，最多3行。
3. 字要大，一眼能看清。
4. 不写长商品标题。
5. 不写“宝子们、姐妹们、家人们”。
6. 不突出价格。
7. 不直接写“包过、必过、官方、内部、押题、100%通过”。

## 主图卖点轮换
1. 时间感：这两天考、考前、临考。
2. 结果感：稳一点、少走弯路、抓对重点。
3. 轻松感：别乱刷、少翻半天。
4. 痛点反转：不是不会，是重点没抓对。
5. 专项痛点：扣分罚款、标志标线、易混题。
6. 真实吐槽：科一真的烦，但有规律。
7. 还愿反馈：刚考完，幸好先看重点。

## 主图结构池
1. 成绩截图型：上方考试成绩页面，下方大字标题。
2. 极简大字型：浅色背景，大字痛点。
3. 资料实拍型：资料/打印纸/学习桌场景。
4. 手机截图感：像手机页面、备忘录、文件列表。
5. 专项重点型：扣分罚款、标志题等重点提示。
6. 临考救命型：时间焦虑 + 大字提醒。
