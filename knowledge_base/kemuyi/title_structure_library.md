# 科目一标题结构库

标题不要写成说明文，不要像商品标题。

标题要学习小红书爆款点击结构，重点使用以下结构轮换。

## 1. 时间焦虑 + 解决出口
示例：

- 后天考科一，现在别乱刷了

## 2. 卡分痛点 + 反转
示例：

- 科一总差几分？先把这几类题理顺

## 3. 临考焦虑 + 安抚
示例：

- 这两天考科一的，先别慌

## 4. 专项痛点 + 提醒
示例：

- 扣分罚款老混？别靠感觉选

## 5. 反常识钩子
示例：

- 题库多，不代表都要死记

## 6. 结果感 + 方法转向
示例：

- 科一想稳一点，重点别抓错了

## 7. 真实吐槽型
示例：

- 科目一真的烦，但不是没规律

## 禁用重复句式
不要机械重复：

- 科目一总卡80多分？
- 别再死刷题了
- 明天考科目一
- 一晚上过科一
- 80个口诀

可以借鉴结构，但必须换角度重写。
