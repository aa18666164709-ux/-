# 科目一主图提示词生成任务

请读取以下文件：

- knowledge_base/kemuyi/product_card.md
- knowledge_base/kemuyi/image_agent_rules.md
- knowledge_base/kemuyi/cover_structure_library.md
- knowledge_base/kemuyi/forbidden_words.md

## 任务
根据每篇笔记的标题和主图文案，生成小红书3:4竖版主图提示词。

## 每条主图提示词必须包含
1. 主图定位
2. 主图结构
3. 主图文案
4. 画面排版建议
5. 完整生图提示词
6. 风险检查结果

## 要求
1. 主图文案短、直给，默认2行，最多3行。
2. 不写长商品标题。
3. 不出现包过、必过、官方、押题、100%通过等风险词。
4. 不要水印、二维码、页码。
5. 不要电商详情页感。
6. 像小红书爆款封面。
